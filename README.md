# 🦞 WeClaw Agent

如果你想尝试在微信发一句话，会有Agent自动帮你执行任务，那么就尝试一下这个吧。
使用了微信文件传输助手作为指令传输的媒介，不会有任何封控的风险。
作为入门Agent最方便的尝试，只需要把文件夹拖到(`chrome://extensions/`)，再登录到网页版微信文件传输助手(`https://filehelper.weixin.qq.com/`)，就能够执行你的操作。
任何你需要操作的网站也需要同步登录，例如知乎、微博、X。


## 💡 这个项目的特点


- **双模型混合双打**：MiniMax 负责当“策划”（意图识别+宏观规划），DeepSeek 负责当“执行”（审阅页面+微观点击）。分工明确，专治各种大模型偷懒。
- **真视智商锁 (IQ Lock)**：大模型有时候会“脑干缺失”，比如把输入框当成发布按钮点。WeClaw 加入了强制拦截机制，一旦发现它犯蠢，立刻打回重做。
- **专治“幽灵输入”**：遇到那种“明明注入了文字，但按钮还是灰的”的前端组件怎么办？我加了底层的事件轰炸和 `composed` 穿透，强行唤醒 Vue/React 的内部状态。
- **真人防风控模拟**：不搞绝对中心点点击，不搞 0 毫秒连发。WeClaw 按按钮之前会悬停、会手抖（坐标微调偏移）、会模拟按压延迟，主打一个“伪装人类”。



## 🚀 怎么跑起来？

1. 把这堆代码 `git clone` 到本地。
2. 打开 Chrome 的扩展管理 (`chrome://extensions/`)，右上角打开**开发者模式**。
3. 点击**加载已解压的扩展程序**，选中这个文件夹。
4. 点开浏览器右上角的插件图标，在设置里填入你的 API Key（默认接的是 SiliconFlow 的接口）。
5. 在侧边栏输入：“我想在小红书发一条图文，今天天气真不错”，然后松开双手，看它表演。
